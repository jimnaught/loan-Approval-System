# Loan Approval System - README

## Overview
This project is a web-based loan approval system that uses machine learning (Random Forest algorithm) to predict loan eligibility based on customer information such as income, dependents, age, and other relevant factors. The system is built with Django as the backend framework.

## Features
- **Machine Learning Integration**: Uses Random Forest algorithm for loan approval predictions
- **User Input Collection**: Gathers customer information including:
  - Income details
  - Number of dependents
  - Age
  - Employment information
  - Loan amount requested
  - Credit history
- **Decision Explanation**: Provides reasoning behind approval/rejection decisions
- **Responsive Interface**: Works across different device sizes

## Technology Stack
- **Backend**: Django (Python)
- **Frontend**: HTML, CSS, JavaScript
- **Machine Learning**: Scikit-learn (Random Forest Classifier)
- **Database**: SQLite (default Django DB)
- **Version Control**: GitHub

## Installation

### Prerequisites
- Python 3.7+
- pip
- virtualenv (recommended)
